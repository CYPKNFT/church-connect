## Color Palette Reference

```
Background:       #0E1116 (Deep Navy)
Cards:            #1A1F26 (Darker Navy)
Primary Accent:   #EACB56 (Gold)
Success/Positive: #2EA98A (Emerald)
Text Primary:     #F4F4F4 (Soft White)
Text Secondary:   #A0A6AE (Cool Gray)
```

## Interactive States

### Tab Navigation
- **Inactive**: Gray text (#A0A6AE)
- **Hover**: White text (#F4F4F4)
- **Active**: Gold text (#EACB56) + gold underline with glow

### Progress Bars
- **Donations**: Gold gradient (#EACB56 â†’ #d4b647)
- **Volunteers**: Emerald gradient (#2EA98A â†’ #26967d)
- **Background**: Semi-transparent white (#F4F4F4/10)

### Buttons
- **Primary (Donate)**: Gold bg (#EACB56), dark text (#0E1116)
- **Success (Volunteer)**: Emerald bg (#2EA98A), white text
- **Secondary**: Outlined, white text with transparent bg

### Cards
- **Border**: Semi-transparent white (#F4F4F4/10)
- **Hover**: Gold border (#EACB56/30)
- **Background**: Dark navy with slight transparency

## Responsive Breakpoints

- **Mobile**: < 768px
  - Single column layouts
  - Horizontal scrolling tabs
  - Fixed bottom CTA bar visible
  
- **Tablet**: 768px - 1024px
  - 2-column grids where applicable
  - Tab bar still horizontal
  
- **Desktop**: > 1024px
  - Full 3-column layouts (Overview tab)
  - 4-column grids (Impact stats)
  - Hero CTA buttons visible

## Animation Timing

- **Tab Switch**: 500ms fade-in
- **Sticky Bar**: 300ms transition
- **Hover Effects**: 200-300ms
- **Progress Bars**: 500ms fill animation
- **Image Hover**: 500ms scale + 300ms overlay
